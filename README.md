Description
This repository contains Python scripts designed to automate repetitive tasks for network engineers and architects. These tools focus on accelerating network discovery, streamlining configuration changes, and validating state across multi vendor environments.

Core Features
Automated Discovery: Inventory collection using Netmiko and Scrapli.

State Validation: Pre and post change verification to ensure network stability.

Config Generation: Jinja2 templating for standardized device deployments.

Audit Tools: Scripts to identify configuration drift and security non compliance.
